# CHATBOT Joby

## Requisitos
1. Anaconda3
2. Librería OpenIA
3. Librería Gradio
4. Librería Requests
5. Librería FastAPI
6. Librería uvicorn

## Instalación de librerías
Para poder instalar los paquetes es necesario ejecutar los siguientes comandos:

    pip install openai
    pip install gradio
    pip install request
    pip install fastapi
    pip install uvicorn

## Ejecución
El siguiente comando permite ejecutar el aplicativo en segundo plano.

    python gradioappgh.py &
